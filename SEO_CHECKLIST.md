# SEO Post-Deployment Checklist
## Doctor Mold Pro - Post-Launch SEO Tasks

---

## Immediate Actions (Within 24 Hours)

### 1. Google Search Console Setup
- [ ] Verify domain ownership in Google Search Console
- [ ] Submit sitemap.xml: `https://doctormoldpro.com/sitemap.xml`
- [ ] Request indexing for homepage
- [ ] Check for crawl errors
- [ ] Monitor Core Web Vitals report

### 2. Google Business Profile (GBP)
- [ ] Claim/verify Google Business Profile listing
- [ ] Ensure NAP (Name, Address, Phone) matches exactly across:
  - Website
  - Google Business Profile
  - BBB profile
  - Social media profiles
- [ ] Add business hours (24/7 emergency)
- [ ] Add service areas (Tampa, St. Pete, Clearwater, etc.)
- [ ] Add photos (van, team, certifications)
- [ ] Enable messaging if desired
- [ ] Add website URL
- [ ] Request reviews from satisfied customers

### 3. Business Information Verification
- [ ] Update `src/config/business.ts` with actual:
  - Full street address (if different from current)
  - Exact GPS coordinates
  - Verified business hours
  - Actual domain URL (replace doctormoldpro.com placeholder)
- [ ] Verify phone number format consistency
- [ ] Ensure address matches across all platforms

---

## Week 1 Tasks

### 4. Local Citations
- [ ] Submit to local directories:
  - Yelp
  - Yellow Pages
  - Angi (formerly Angie's List)
  - HomeAdvisor
  - Thumbtack
- [ ] Ensure NAP consistency across all citations
- [ ] Add website URL to all profiles

### 5. Schema Validation
- [ ] Test JSON-LD schema using:
  - [Google Rich Results Test](https://search.google.com/test/rich-results)
  - [Schema.org Validator](https://validator.schema.org/)
- [ ] Fix any schema errors
- [ ] Verify LocalBusiness schema appears in search results

### 6. Analytics & Tracking
- [ ] Verify Google Analytics is tracking correctly
- [ ] Set up conversion goals:
  - Phone call clicks (tel: links)
  - Form submissions
  - PDF downloads
- [ ] Set up Google Tag Manager (if using)
- [ ] Monitor initial traffic and behavior

---

## Week 2-4 Tasks

### 7. Content & Backlinks
- [ ] Create Google Business Profile posts (weekly)
- [ ] Share on social media (Facebook, LinkedIn)
- [ ] Consider local PR opportunities
- [ ] Reach out to local Tampa business associations
- [ ] Consider guest posting on Tampa home improvement blogs

### 8. Review Strategy
- [ ] Set up review request process
- [ ] Add review request to post-service follow-up
- [ ] Monitor and respond to all reviews (positive and negative)
- [ ] Display reviews prominently on website

### 9. Performance Monitoring
- [ ] Run Lighthouse audits weekly
- [ ] Monitor Core Web Vitals in Search Console
- [ ] Check mobile usability report
- [ ] Monitor keyword rankings (use free tools like Ubersuggest or Google Search Console)

---

## Ongoing Maintenance

### Monthly
- [ ] Update sitemap.xml lastmod dates if content changes
- [ ] Review and respond to Google Business Profile reviews
- [ ] Check for broken links
- [ ] Monitor search console for errors
- [ ] Review analytics for conversion improvements

### Quarterly
- [ ] Audit content for freshness
- [ ] Update service area information if expanded
- [ ] Review and update certifications/licenses
- [ ] Check competitor SEO strategies
- [ ] Review and optimize underperforming pages

---

## Key Metrics to Track

### SEO Metrics
- Organic search impressions
- Click-through rate (CTR)
- Average position for target keywords:
  - "mold testing Tampa"
  - "mold removal Tampa"
  - "mold remediation Tampa"
  - "mold inspection Tampa FL"

### Local SEO Metrics
- Google Business Profile views
- Direction requests
- Phone calls from GBP
- Review count and average rating

### Technical Metrics
- Lighthouse scores (aim for 90+)
- Core Web Vitals (LCP, FID, CLS)
- Mobile usability score
- Page load speed

### Conversion Metrics
- Form submissions
- Phone call clicks
- PDF downloads
- Time on page
- Bounce rate

---

## Important Notes

1. **NAP Consistency:** Ensure Name, Address, Phone are identical everywhere
2. **Domain:** Update all placeholder URLs (doctormoldpro.com) with actual domain
3. **Schema:** Keep JSON-LD schema updated as business info changes
4. **Mobile First:** Always test changes on mobile devices
5. **Speed:** Monitor page speed - aim for <3s load time
